Information Reconciliation Library for CV-QKD Systems
=================================================================================

+++++++++++++++++++++++++++++++++++++++++++++
Overview
+++++++++++++++++++++++++++++++++++++++++++++

Welcome to the official documentation for the Information Reconciliation Library for Continuous-Variable Quantum Key Distribution (CV-QKD) Systems. This comprehensive library provides a suite of tools designed for reconciling measured states in CV-QKD systems.

Developed in C++, this library offers a robust set of utilities to facilitate the reconciliation process. Its versatile design allows for integration across multiple platforms:

* Direct implementation in C++
* Python integration via provided bindings
* MATLAB compatibility through Python bindings
